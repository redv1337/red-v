## Bankautomaten
Es ist nun möglich, Bankautomaten im Spiel zu benutzen.
Dazu musst du einfach an einen Automaten gehen dein Inventar (I) öffnen und auf der EC-Karte -> Benutzen drücken.
Danach taucht ein Bankmenü auf.
